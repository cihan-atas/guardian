Hedef sunucuya guardian-agent'ı kur. Sertifika sunucu tarafından otomatik imzalanır (enrollment). Kayıt token'ı 30 dk geçerli, tek kullanımlıktır.

Sunucu kayıtlı değilse önce Sunucular ekranından ekleyin.

Aşağıdaki komutu hedef sunucuda root/sudo ile çalıştırın. Script; kullanıcı/dizin/sertifika/systemd dahil her şeyi kurar.

{{ enroll.install_command }}

Token {{ enroll.expires_at | date:'HH:mm:ss' }}'e kadar geçerli. Kurulum sonrası sshd ayarları + kullanıcı authorized_keys adımları script çıktısında listelenir.

Sunucuda agent binary bulunamadı; script binary'yi indiremez. Sunucuda GUARDIAN_AGENT_BINARY_PATH ayarlayın.
